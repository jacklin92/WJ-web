# WJ-web

個人 3D 作品集與技術部落格，使用 Astro + React + Three.js 打造。

## 技術棧

- Astro (SSG)
- React 18 + TypeScript
- Three.js / React Three Fiber
- Tailwind CSS v4
- Markdown / MDX

## 常用語言

- Python
- Jupyter Notebook
- Astro
- MDX

## 指令

| 指令 | 說明 |
| :--- | :--- |
| `npm install` | 安裝依賴 |
| `npm run dev` | 啟動本地開發伺服器 (localhost:4321) |
| `npm run build` | 建置正式環境至 `./dist/` |
| `npm run preview` | 本地預覽建置結果 |

## 部署

推送至 `main` 分支後，GitHub Actions 會自動建置並部署至 GitHub Pages。
